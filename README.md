# Omar Abdelrahman's Portfolio

A professional portfolio website showcasing my skills, experiences, projects, and achievements.

## Overview

This portfolio is built using HTML, CSS, and JavaScript to create a responsive, modern, and interactive user experience. It provides visitors with:

- An overview of my professional background and skills
- Detailed information about my work experience
- Showcase of my latest projects with links to live demos and repositories
- Educational background and certifications
- Notable achievements and awards
- Contact information and a contact form for reaching out

## Features

- **Responsive Design**: Fully responsive layout that works on mobile devices, tablets, and desktop screens
- **Modern UI**: Clean and professional design with smooth animations and transitions
- **Interactive Elements**: Navigation menu, skill bars, project cards, and contact form
- **Section-Based Layout**: Organized into distinct sections for easy navigation
- **Social Media Integration**: Links to LinkedIn, GitHub, LeetCode, Codeforces, Medium, and other professional profiles

## Technologies Used

- HTML5 for structure
- CSS3 for styling and animations
- JavaScript for interactivity
- Font Awesome for icons
- Google Fonts for typography

## Sections

1. **Hero Section**: Introduction and quick summary of my professional focus
2. **About Section**: Detailed professional overview and personal information
3. **Skills Section**: Technical skills categorized by domains
4. **Experience Section**: Professional work history with detailed descriptions
5. **Projects Section**: Showcase of latest projects with links and technologies used
6. **Education Section**: Academic background and certifications
7. **Achievements Section**: Notable accomplishments and recognitions
8. **Contact Section**: Contact information and form for direct messaging

## Setup and Usage

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the different sections to explore my portfolio

## Local Development

To modify the portfolio:

1. Edit `index.html` to update content
2. Modify `styles.css` to change styling
3. Update `script.js` to add or modify interactive features
